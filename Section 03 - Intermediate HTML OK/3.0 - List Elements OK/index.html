Ramzan's Recipe for Best
Chicken Biryani

Ingredients

For the Rice:

  • Rice Half Kg
  • Chicken Meat Half KG
  • Shan Biryani Masala
  • Spices
  • Hot Water

For more Taste:

  • Cook on wood
  • Add Haleeb Cream

Instructions:

  1. Boil the water
  2. Put Rice into it
  3. Add Shan Biryani masala
  4. Add Spices
  5. Add Haleeb Cream
